Subtraction using Two’s Complement with 4-bit Binary Parallel Adder (Verilog)

This project implements binary subtraction using the two’s complement method with a 4-bit binary parallel adder in Verilog HDL.
The design converts the subtrahend into its two’s complement form and performs subtraction by adding it to the minuend using a 4-bit ripple carry/parallel adder architecture.

The project demonstrates fundamental digital arithmetic operations used in ALUs and highlights how subtraction can be efficiently implemented using addition circuits in digital systems.
A testbench is included to verify different input combinations and validate the correctness of the subtraction operation through simulation waveforms.

Key Features

Implementation of two’s complement based subtraction

4-bit binary parallel adder architecture

Designed using Verilog HDL

Includes testbench for functional verification

Simulation results demonstrating correct subtraction operation
